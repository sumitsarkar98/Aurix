# Aurix

Aurix is a modern **full-stack MERN** application designed to streamline **gold purchase management**. It provides businesses with a secure, efficient, and user-friendly platform to manage customers, purchase transactions, and real-time gold rates through a centralized dashboard.

---

# ✨ Features

## 🔐 User Authentication

- Secure user registration (Sign Up)
- User Login
- User Logout
- JWT-based authentication
- Protected routes

---

## 💰 Real-Time Gold Price

- Display the current day's gold price
- Securely fetch and manage gold rates
- Automatically apply a **3% margin adjustment** before displaying and calculating prices

---

## 🧮 Gold Price Calculator

- Enter gold weight
- Select gold purity (24K / 22K / 18K)
- Instantly calculate the estimated payout amount
- Uses the latest adjusted gold rate for accurate calculations

---

## 📍 Lead Submission

- Submit customer transaction details
- Automatically capture the user's current GPS location
- Store customer leads securely in the database

---

## 👨‍💼 Admin Portal

- View all submitted customer leads
- Update lead status
- Manage customer follow-ups
- Monitor transactions from a centralized dashboard
